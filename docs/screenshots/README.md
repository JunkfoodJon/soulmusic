# Screenshots

This directory contains screenshots of the SoulMusic GUI for documentation purposes.

## How to Contribute Screenshots

Screenshots help new users understand the interface before installing. If you have
SoulMusic running, please capture and submit screenshots of each major tab.

### Recommended Screenshots

| Filename | Description |
|----------|-------------|
| `dashboard-overview.png` | Full application window showing the main dashboard |
| `tab-signal-analysis.png` | Signal Analysis tab with a live or recorded signal loaded |
| `tab-beamforming.png` | Beamforming Visualisation tab |
| `tab-platform-id.png` | Platform Identification tab with a detected platform |
| `tab-emitter.png` | Emitter / Waveform Generation tab |
| `tab-module-loader.png` | Module Loader tab with one or more plugins loaded |

### Capture Guidelines

1. Use a **1280 × 800** or **1920 × 1080** resolution for consistency.
2. PNG format, lossless compression.
3. Do **not** include real operational data, personally identifiable information,
   or actual drone targets in screenshots.
4. Synthetic / simulated signal data is fine.

### How to Add

1. Place your `.png` files in this directory.
2. Update the root `README.md` to uncomment and fill in the `![…](docs/screenshots/…)` lines.
3. Open a pull request with a description of what the screenshot shows.
